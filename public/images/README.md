# Background Images

Add your background photos here with the following names:

- `hero-bg.jpg` - Main landing page background (Charles River or Golden Gate Bridge)
- `about-bg.jpg` - About section background
- `skills-bg.jpg` - Skills section background
- `projects-bg.jpg` - Projects section background
- `contact-bg.jpg` - Contact section background

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: At least 1920x1080px (Full HD) for best quality
- **File size**: Optimize images to keep them under 500KB each for faster loading
- **Orientation**: Landscape works best for full-screen backgrounds

## Tips

- Use high-quality photos of the Charles River and Golden Gate Bridge
- Make sure images have good contrast for text readability
- Consider the dark overlay (60-70% opacity) when choosing images
- Test on both desktop and mobile to ensure images look good at all sizes

## Image Optimization

You can use tools like:
- [TinyPNG](https://tinypng.com/) - Compress images
- [Squoosh](https://squoosh.app/) - Google's image optimizer
- Next.js Image component (for future optimization)
